Extended Facet Amalgamation Approach (FAA)

Digital Mapping and Kinematic Analysis of Rock Slopes
GEO Technical Note TN 4/2024 · K.K. Chan · Planning and Development Division

What is FAA?

The Facet Amalgamation Approach is a novel technique for extracting major joint sets from the point cloud of a scanned rock slope. Unlike existing methods that require extensive parameter tuning, FAA performs kinematic analysis directly on each point or triangle without first delineating individual discontinuity planes. Results are mapped back to the 3-D model so that potentially unstable zones are immediately visible.

Failure Modes

Planar Sliding

Discontinuity daylights on slope face; dip exceeds friction angle; dip direction within lateral limits.

Flexural Toppling

Sub-vertical joints dip steeply into the slope; compressive stress overcomes friction at the base.

Wedge Failure

Intersection of two converging discontinuities daylights on the slope; line of intersection is outside friction cone.

Workflow

  1. Acquire 3-D point cloud or mesh via LiDAR / UAV photogrammetry
  2. Pre-process: register, de-noise, estimate normals (CloudCompare)
  3. Load into this app; set slope orientation and geotechnical parameters
  4. Run Sliding / Toppling / Wedge analyses
  5. Review coloured results; export for further assessment or BIM integration
